# 📖 故事导演-考研版 (Story Director - Postgraduate English Exam Edition)

这是一个基于 DeepSeek API 驱动的文字冒险（Text Adventure）RPG 游戏。本项目专为备战 **考研英语** 的考生量身定制，在提供沉浸式互动游戏体验的同时，利用 AI 生成包含考研级别高级词汇、长难句和正式表达的故事文本，帮助玩家在娱乐中无痛提升英语阅读理解能力。

## ✨ 核心特性

- 🧠 **考研英语专属语料**: 提示词经过精心调优，AI 叙述将默认使用考研级别的高级词汇（约 7000 词大纲）和复杂句式，完美契合考研英语阅读的行文风格。
- 🎮 **经典 RPG 机制**:
  - **属性系统**: 包含生命值 (HP)、精力值 (ENG)、幸运值 (LUCK) 和金币 (GOLD)。你的每一个选择都可能影响这些属性。
  - **背包系统**: 在探索过程中收集、使用物品（如药水、钥匙、地图等）来改变故事走向。
- 📖 **逐句精读模式**: 为攻克长难句专门设计的阅读辅助功能。游戏会高亮当前阅读的句子，支持“上一句/下一句”切换，让你有充足的时间分析长难句的句子结构。
- 💾 **进度管理**: 
  - 支持将完整的故事日志导出为 TXT 文本，方便后续作为阅读材料复盘和复习。
  - 支持游戏存档与读档，随时随地继续你的冒险。
- 🖥️ **轻量级 GUI**: 基于 Python 内置的 `tkinter` 构建，无需繁琐的环境配置，界面简洁清爽。

## 🚀 快速开始

本项目提供两种运行方式，你可以根据自己的需求选择：

### 方式一：直接运行免安装版 (推荐给普通玩家)
如果你不想安装 Python 环境，可以直接使用打包好的独立可执行文件：
1. 在项目的 [Releases] 页面下载最新的 `StoryDirector.exe`。
2. 将其放置在一个单独的文件夹中（游戏存档会保存在该文件夹下）。
3. 双击运行即可开始游戏。
> **⚠️ 注意**：由于游戏为了安全不会在本地储存你的密钥，因此**每次启动游戏时**都会弹窗提示输入你的 [DeepSeek API Key](https://platform.deepseek.com/)。如果你想一劳永逸，推荐使用下方“方式二”将密钥写入源码后自行打包。

### 方式二：通过 Python 源码运行 (推荐给开发者/想自定义的玩家)
**环境要求：** Python 3.7 或更高版本。
本项目仅使用 Python 标准库（`tkinter`, `urllib`, `json` 等），**无需** `pip install` 任何第三方依赖。

1. 克隆或下载本仓库到本地。
2. **API Key 配置 (可选但推荐)**: 用代码编辑器打开 `main.py`，找到文件顶部的 `DEEPSEEK_API_KEY = ""`，将你的 API Key 填入双引号中。这样每次运行就**不需要**再手动输入了。
3. 在项目根目录下运行主程序：
   ```bash
   python main.py
   ```

## 🕹️ 怎么玩？

1. **新游戏**: 点击 "New Game"，输入一个你感兴趣的冒险主题（例如 "Cyberpunk City", "Magic School" 或是 "Abandoned Hospital"）。
2. **阅读与分析**: AI 会根据主题生成一段考研难度的背景故事。你可以使用底部的“上一句/下一句”按钮逐句精读文本。
3. **做出选择**: 阅读完毕后，系统会提供 2-3 个行动选项。请结合你的当前状态（HP、ENG 等）做出最明智的选择。
4. **使用物品**: 点击右侧 Inventory (背包) 中的物品，可以查看物品描述，并在合适的时机点击 "Use" 使用它来化解危机。
5. **生存下去**: 注意保持你的 Health (生命值) 和 Energy (精力值) 大于 0，否则游戏结束！

## 🛠️ 进阶开发与自定义配置

### 自定义游戏难度与风格
如果你想调整游戏的难度、AI 的创造力或初始属性，可以直接修改 `main.py` 中的以下常量：
- `SYSTEM_PROMPT`: 调整 AI 的角色设定、词汇限制和输出语言风格。
- `DEFAULT_STATS`: 调整玩家开局的初始血量、精力和金币。
- 在 `DeepSeekClient` 类的 `_send_request` 方法中，可以修改 API 请求的 `temperature` 参数以控制故事的随机性和发散程度（当前设定为较高的 1.2 以增强故事趣味性）。

### 📦 如何自行打包为 .exe 独立程序
如果你修改了源码（例如填入了你自己的 API Key），并希望自己打包成 `.exe` 分享给同学，请按照以下步骤操作：

1. **修改路径逻辑（非常重要）**：
   在打包前，必须修改 `main.py` 第 19 行关于 `SCRIPT_DIR` 的定义，否则打包成单文件后，游戏导出的存档会丢失。请将其替换为：
   ```python
   import sys
   if getattr(sys, 'frozen', False):
       SCRIPT_DIR = os.path.dirname(sys.executable)
   else:
       SCRIPT_DIR = os.path.dirname(os.path.abspath(__file__))
   ```
2. **安装打包工具**:
   ```bash
   pip install pyinstaller
   ```
3. **执行打包命令**:
   在项目目录下运行以下命令（`-F` 表示打包为单文件，`-w` 表示隐藏控制台黑框）：
   ```bash
   pyinstaller -F -w main.py -n "StoryDirector"
   ```
4. 打包完成后，即可在生成的 `dist` 文件夹中找到 `StoryDirector.exe`。此时的 exe 已经内置了你的配置，直接双击即可畅玩！
