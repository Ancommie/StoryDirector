# 📖 故事导演-考研版 (Story Director - Postgraduate English Exam Edition)

这是一个基于 DeepSeek API 驱动的文字冒险（Text Adventure）RPG 游戏。本项目专为备战 **考研英语** 的考生量身定制，在提供沉浸式互动游戏体验的同时，利用 AI 生成包含考研级别高级词汇、长难句和正式表达的故事文本，帮助玩家在娱乐中无痛提升英语阅读理解能力。

## ✨ 核心特性

- 🧠 **考研英语专属语料**: 提示词经过精心调优，AI 叙述将默认使用考研级别的高级词汇和复杂句式，完美契合考研英语阅读的行文风格。
- 🎮 **经典 RPG 机制**:
  - **属性系统**: 包含生命值 (HP)、精力值 (ENG)、幸运值 (LUCK) 和金币 (GOLD)。你的每一个选择都可能影响这些属性。
  - **背包系统**: 在探索过程中收集、使用物品（如药水、钥匙、地图等）来改变故事走向。
- 📖 **逐句精读模式**: 为攻克长难句专门设计的阅读辅助功能。游戏会高亮当前阅读的句子，支持“上一句/下一句”切换，让你有充足的时间分析句子结构。
- 💾 **进度管理**: 
  - 支持将完整的故事日志导出为 TXT 文本，方便后续作为阅读材料复习。
  - 支持游戏存档与读档，随时随地继续你的冒险。
- 🖥️ **轻量级 GUI**: 基于 Python 内置的 `tkinter` 构建，无需繁琐的环境配置，开箱即用。

## 🚀 快速开始

### 环境要求
- Python 3.7 或更高版本
- 只需要 Python 标准库（`tkinter`, `urllib`, `json` 等），**无需** `pip install` 任何第三方包。

### 运行游戏
1. 克隆或下载本仓库到本地。
2. 在项目根目录下运行主程序：
   ```bash
   python main.py
   ```
3. **API Key 配置**: 游戏启动后，会提示你输入 [DeepSeek API Key](https://platform.deepseek.com/)。你也可以直接在代码中修改 `DEEPSEEK_API_KEY` 常量，免去每次输入的麻烦。

## 🕹️ 怎么玩？

1. **新游戏**: 点击 "New Game"，输入一个你感兴趣的冒险主题（例如 "Cyberpunk City" 或 "Magic School"）。
2. **阅读与分析**: AI 会根据主题生成一段考研难度的背景故事。你可以使用底部的“上一句/下一句”按钮逐句精读。
3. **做出选择**: 阅读完毕后，会显示 2-3 个行动选项。结合你的当前状态（HP、ENG等）做出最明智的选择。
4. **使用物品**: 点击右侧 Inventory (背包) 中的物品，可以查看物品描述并选择在合适的时机使用它。
5. **生存下去**: 注意保持你的 Health (生命值) 和 Energy (精力值) 大于 0，否则游戏结束！

## 🛠️ 自定义配置

如果你想调整游戏的难度、AI 的创造力或初始属性，可以修改 `main.py` 中的以下常量：
- `SYSTEM_PROMPT`: 调整 AI 的角色设定和输出语言风格。
- `DEFAULT_STATS`: 调整玩家开局的初始血量、精力和金币。
- 可以在 `DeepSeekClient` 类中修改 `temperature` 参数以控制故事的随机性和发散程度（当前设定为较高的 1.2 以增强故事趣味性）。
